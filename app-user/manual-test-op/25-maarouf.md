# اختبار معروف - المفقودات والموجودات

## نظرة عامة

هذا الملف يغطي اختبار عمليات إدارة إعلانات المفقودات والموجودات.

---

## العمليات

### 1. إنشاء إعلان مفقود أو موجود

**Endpoint:** `POST /maarouf`

**Request Body:**
```json
{
  "ownerId": "507f1f77bcf86cd799439011",
  "title": "محفظة سوداء مفقودة في منطقة النرجس",
  "description": "محفظة سوداء صغيرة تحتوي على بطاقات شخصية وأموال",
  "kind": "lost",
  "tags": ["محفظة", "بطاقات", "نرجس"],
  "metadata": {
    "color": "أسود",
    "location": "النرجس",
    "date": "2024-01-15"
  }
}
```

### 2. قائمة الإعلانات

**Endpoint:** `GET /maarouf`

### 3. تفاصيل إعلان

**Endpoint:** `GET /maarouf/:id`

### 4. تحديث إعلان

**Endpoint:** `PATCH /maarouf/:id`

### 5. حذف إعلان

**Endpoint:** `DELETE /maarouf/:id`
